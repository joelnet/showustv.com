# Similar / related projects

Show Us TV is a TV-and-movie tracker in the spirit of TV Time. A few other projects tackle the same "track what you watch" problem and are worth a look:

- **[Yamtrack](https://github.com/FuzzyGrim/Yamtrack)** — a self-hostable, open-source media tracker (TV, movies, anime, manga, games). A good open-source project in the same space.
- **TV Time** — the popular hosted show tracker (shutting down), and the habit most of our users are migrating from.

If you know of another project that belongs here, open a PR adding it to this list.
